O que significa HTML?

(Voltar)

O HTLM (Hypertext Markup Language) é uma linguagem de hipertexto e marcação, e esta é sua característica principal, o que torna tão poderosa para o desenvolvimento de sites. As marcações são enviadas para os olhos nos textos, porem os olhos dos leitores elas são invisíveis, pois sua finalidade principal é de instruções ao navegador para que o mesmo possa ser uma paginação.

O que são Tags

(Voltar)

As tags também são responsáveis por criar elementos nas paginas web com a função de organizar, e limitar o conteúdo de uma pagina. Cabe então ao Browser reconhecer esses elementos, e renderizá-las apresentando seu conteúdo aos usuários.

A importancia das Tags

(Voltar)

As tags são muito importantes e fundamentais para o desenvolvimento WEB, e usa-las de forma correta vai agregar bastante para o seu site, principalmente em relação ao SEO . É através das tags que se classificam os tópicos do site e definem o que será agrupado entre os diversos conteúdos da pagina de forma temática e organizada.

Diferença entre lista ordenada e não-ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados, a lista ordenada enumera os elementos, já a lista não-ordenada coloca marcadores antes de cada elemento.
